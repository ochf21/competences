SAE5.01 ‚Äì Design, Build and Present a Technical Solution

Skill: IoT Solutions & Embedded Systems

Project: Vehicle Tracking by GNSS

Design and implementation of an embedded geolocation system using IoT technologies for vehicle tracking with a LoRaWAN architecture.

Main Objective

Use the Pycom environment to transmit GNSS data from a vehicle through The Things Network (TTN) LoRaWAN network and visualize its position on a map via a web server.

Final Goal: Low-Power Autonomous System

  • Remain in sleep mode (deep sleep) by default
  • Wake up automatically when motion is detected (accelerometer)
  • Transmit geographic position via LoRaWAN
  • Send SMS alerts for unauthorized movement
  • Enter sleep mode remotely via web interface
